the Details server side : TFTP

  • Trivial file transfer protocol

  • Allows DC to fetch the specified Boot file from server

  • Provided as an inetd daemon

  • Enabled by editing /etc/inetd.conf

  • Directories to export provided as parameters in the inetd.conf file [default /tftpboot]

  • Tagged image file containing the kernel is placed in this directory
