the Details DC related : NFS (2)

Bin, sbin, lib and boot

  • Make a copy of the original Server directories at say /tftpboot/backup

  • Create HardLinks from the individual /tftpboot/ip-addr directories to these

  • This makes them available to DC as soon as it loads its root fs using NFS. Essential as these contain the files accessed during the init process

    • cp -a bin sbin lib boot dev etc var /tftpboot/192.168.2.129
    • ln /tfptboot/192.168.2.129/lib/* /tftpboot/192.168.2.130/lib
