  • Provides the DC with its root filesystem over the network

  • Started as part of one of the rc files

  • Files to export configured in /etc/exports

    • Directories to export
    • List of m/cs that are allowed access

  • Run exportfs to sync the nfsd's memory list with the /etc/exports file
