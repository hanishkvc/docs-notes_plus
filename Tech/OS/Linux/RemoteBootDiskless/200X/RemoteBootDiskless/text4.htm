Whats involved

  • Server which can store the OS and Applications

  • Diskless client (DC) with Netloader on EPROM or floppy.

  • Mechanism for DC to dynamically get its IP and network info [BOOTP,RARP]

  • Fetching of Kernel and related files over network [TFTP]

  • Sharing of files over the network [NFS]
