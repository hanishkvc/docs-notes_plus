the Details DC related : Kernel image (2)

  • Netbootimage of kernel

    • Net loaders use a TAGGED Image file so that one can pass a lot of content and info to it from the server.
      • One of its contents is the Kernel
    • Etherboot and Netboot come with mknbi-linux which can create this netbootimage
      • Remember to set the path to BCC's include and lib files when compiling these if required
    • Copy this file to the TFTP export directory
